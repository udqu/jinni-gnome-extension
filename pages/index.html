Jinni for GNOME Shell

About

Jinni is a lightweight task manager app for the GNOME Shell Top Bar. The orginal app is developed by Dino Angelov and available for OS X 10.8 or later (Intel only): https://mac.softpedia.com/get/Utilities/Jinni.shtml

Installation

To install the extension, clone the repository and copy the `jinni-notes@yourdomain` directory to `~/.local/share/gnome-shell/extensions/`.

Usage

Access Jinni Notes from the GNOME Shell Top Bar to quickly create and manage your notes.
